Konventionen für diese Richtlinien

In den Konventionen werden einzelne Begriffe, Formatierungen, bestimmte Verwendungsweisen definiert und erklärt, die im Text der Richtlinien angewendet werden.

Schriftarten
Die Verwendung der Nichtproportionalschrift erfolgt bei:
  • der Angabe von Transkriptionsbeispielen,
  • der Angabe von Tags bzw. Codebeispielen.
Unicode
Die Notation U+NNNN verweist auf ein entsprechendes Unicode-Zeichen. Desweiteren wird in den Transkriptionsbeispielen bei höherbittigen Unicode-Zeichen die von XML abgeleitete Notationsform &#xNNNN; benutzt.
Vorlage
Unter der Vorlage sind, so nicht anders vermerkt, die digitalen Faksimiles einer Buchausgabe zu verstehen.
Herausgegeben von OCR-D.

Die Richtlinien zur Transkription für Ground Truth, basieren auf OCR-D Guidelines/2.0.0. OCR-D GitHub
